Новая программа ГеоКонструктор 1.2 
Пока проходит активное тестирование и является альтернативной заменой предыдущим версиям Конвертера. 
Основные отличия: 

Увеличена скорость обработки данных; 
Решена проблема внутренних полигонов (сложные водные объекты с островами и т.п.); 
Упрощена процедура конвертации (Open File > Save as); 
Появилась визуализация объектов; 
Обрабатываются 3D развязки; 
Включена поддержка меню «Места интереса» на английском языке; 
Появилась возможность редактирования объектов, подгрузки растров; 
Появилась возможность выбора пользователем скоростного режима дорог (с учётом или без RouteClass), а так же другие настройки; 
Решена проблема адресного поиска для версий CityGuide 3.7. и выше. 
Адреса используются не только из категорий Building, но и из пои. 
Появилась полная поддержка в исходных данных кодировки UTF-8. 
Предназначен не только для конвертирования, но и для редактирования карт (в формате dcf) 


По умолчанию установлены минимальные настройки. При необходимости, можно их изменить. 
1. Настройки при загрузке: 
Converting rules — оставляем всегда тот, что указан по умолчанию; 
Compilation Scale – Масштаб компиляции. 
                                  1:10 000 Town, City – Для карт городов и некрупных областей 
                                  1:100 000 Region – Крупные области и небольшие государства 
                                  1:500 000 Country – Обзорные карты государств 
                                  1:1000 000 Country – Евразия 
Load each layer separately — формирует из каждого непустого масштабного слоя (Level) отдельный dcf файл. 
Load addresses according to the converting rules – использует адресные данные как из категории Building (как в прошлых версиях конвертера), так и из пои. 
Load road speed according to the converting rules – Пересчёт скоростей с учётом RouteClass. Если галочка отсутствует — скорости будут точно такие же как и в исходнике. 
Create intermediate scale levels – Галочка нужна, если карта однослойная. 
Fill RECIND with line number and object type (e.g. [12345] Type=0x1) – Вписывает в атрибут RecordIndication тип-код исходного объекта и номер строки с описание (можно найти, если карту.mp открыть текстовым редактором). Эта галочка на конвертацию не влияет, нужна только для исследования. 
Fill SORIND with all object attributes (e.g. Label=...; RoadID=...; etc.) – Аналогична предыдущей. В атрибут Source Indication вписывает абсолютно всё, что находит в свойствах исходного объекта. 
Load unknown objects as invisible graphic primitives – при снятой галочке игнорирует объекты, которые не входят в состав готовой карты. 

2. Настройки при сохранении: 
Ignore negative road level – галочка нужна, если в исходной карте содержатся отрицательные значения 3D-развязок 
Language – выбор языка, на котором будут прописаны категории в меню «Места интереса» 
Codepage – кодировка исходных данных 



Пакетный Режим:


Параметры командной строки
 
-mp:<имя_файла_mp>            этот параметр обязателен для конвертации MP в DCM
-dcm:<имя_файла_dcm>          по-умолчанию берется имя файла MP и расширение меняется на DCM
-loadrule:<имя_файла_shm>     по-умолчанию PolishConverter.shm
-saverule:<имя_файла_shm>     по-умолчанию POICategories.shm
-lang:<акроним_языка>         по-умолчанию RU
-codepage:<номер_страницы>    по-умолчанию берется кодовая страница исходного MP
-scale:<масштаб_компиляции>   по-умолчанию 10000
-keepdcf                      сохранять рабочие файлы DCF, по-умолчанию отключено
-lineaddress                  заружать линейную адреску (из Гармин-формата), по умолчанию отключено
-createlevels                 сформировать многослойную карту, по умолчанию отключено


Имена файлов задаются с полным путем. Если пути нет, то подразумевается, что файл рядом с exe-шником.
